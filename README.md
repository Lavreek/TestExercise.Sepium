#### Краткий обзор
+ `.docker` - докер образ
+ `bundle` - классы помощники
+ `migrations` - таблицы
+ `request` - обрабатываемые запросы
+ `src` - основной пул классов
+ `templates` - кусочки html
+ `var` - для прочего
+ `.env` - конфигурация
